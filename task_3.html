HTML is Hard

HTML is a Hyper Text Transfer Protocol. It is used to create layout or design of a website

Block Elements

Characteristics

  • Start on a New Line: Block elements always start on a new line, which helps in clearly separating different sections of content.
  • Occupy Full Width: By default, block elements stretch to take up the full width of their container, which means they are as wide as the parent element allows.
  • Contain Other Elements: Block elements can contain other block elements as well as inline elements, making them useful for creating complex layouts.

    • Common block elements of HTML

  • <div>:A generic container for content. It doesn't inherently represent anything.
  • <P>: Represents a paragraph of text.
  • <ul>: Represents an unordered (bulleted) list.
  • <Header>: Represents introductory content, typically a group of introductory or navigational aids.

    • Inline Elements

    Characteristics

  • Do Not Start on a New Line: Unlike block elements, inline elements do not start on a new line. They remain in the flow of the surrounding content.
  • Take Up Only Necessary Width: Inline elements only take up as much width as their content requires. They do not stretch to fill the width of their container.
  • Contained Within Block Elements: Inline elements can only exist within block elements. They cannot directly contain block elements, but they can contain other inline elements.

    • Common inline elements of HTML

  • <Span>: A generic inline container for phrasing content.
  • <a>: Defines a hyperlink.
  • <br>: Inserts a line break.
  • <b>: Makes text bold.
